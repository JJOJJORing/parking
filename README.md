# 주차장 관리 프로그램

### 소개
 - 주차장의 입출차 관리와 정산 시스템

### 목적
 - 주차장 관리 시스템을 통해서 장고의 기본적인 MTV패턴과 ORM을 익힌다
 
## 아키텍쳐
- 프론트엔드
    - html5, css3, JS
- 백엔드
    - Django
- 데이터 베이스
    - mysql
